# Bookkeeping Templates

Minimal starter content for the project-index bookkeeping file onboard creates under the
project root. Write it only if it does not already exist. Keep it skeletal — it fills
in as the user works (project-save maintains the index).

---

## `{CLAUDE_PROJECT_ROOT}/context/project-index.md`

```markdown
# Project Index

Manifest of all projects — one row each. Keep the Phase column terse (1-3 words).
project-save maintains this file; do not duplicate detailed phase context here.

| Project | Phase | Last Active | Status | Next Step |
|---------|-------|-------------|--------|-----------|

Status values: Active · Blocked · Backlog · Complete · Archived
```
